#### 一、前端布局标签

以下为一个网页的基础布局

```html
<!DOCTYPE html>
<html lang='zh-CN'>
    <head>
        <meta charset='utf-8'>
        <meta name='keyword' content='html,html5'>
        <meta name='description' content='一个神奇的网站'>
        <title>html5</title>
    </head>
    <body>
        <header>
            头部内容
        	<nav>
                <ul>
                    <li>导航栏</li>
                    <li></li>
                </ul>
            </nav>
        </header>
        <main>
            <aside>侧边栏</aside>
        	<article>
                主要内容
            	<section>相似内容区块</section>
                <section>相似内容取款</section>
            </article>
            <article>
                主要内容
            	<section>相似内容区块</section>
                <section>相似内容取款</section>
            </article>
        </main>
        <footer>
        	底部内容
        </footer>
    </body>
</html>
```
