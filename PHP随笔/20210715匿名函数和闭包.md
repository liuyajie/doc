#### 一、匿名函数的定义

```php
// 匿名函数的定义
$closure = function () {
    echo 2233;
};
$closure();

// 匿名函数传参
$closure = function ($a,$b){
    echo $a + $b;
};
$closure(2,6);

// 匿名函数无法直接使用外部变量，需要引入才可使用
$str = 'abcd';
$closure = function () use ($str) {
    echo $str;
};
$closure();

// 但可以使用类中的公共静态变量和类常量
class Order
{
    const PI = 3.1415926;
    public static $name = 'OrderName';
}

$closure = function () use ($str) {
    echo Order::$name;
    echo Order::PI;
};
$closure();
```

#### 二、匿名函数的使用

1、当做回调函数使用

```php
$data = array_map(function ($value) {
    return $value * $value;
}, [1, 2, 3]);
```

2、当做闭包使用，把函数内部的执行环境保存下来，在其他地方使用。在JS中很常用，需要保存作用域，另作他用。

```php
function sayHello()
{
    $greeting = 'hello';
    return function ($name) use (&$greeting) {
        return $greeting . ':' . $name;
    };
}

$clourse = sayHello();
echo $clourse('liuyajie');
```

3、修改对象的私有属性使用

 	1. 匿名函数也是一种特殊的对象，该对象只有一个方法（bindTo）
 	2. 第一个参数是指定对象，指定之后匿名函数内可以使用$this
 	3. 第二个参数是指定作用域，就是把这段匿名函数的代码插入到类中

```php

class Person
{
    private $name;
    public function getName()
    {
        echo $this->name;
    }
}

// 没有指定作用域，可以使用$this，但不能访问私有属性  报错，无法访问私有属性
$person = new Person();
$clourse = function () {
    $this->name = 'lyj';
};
$clourse1 = $clourse->bindTo($person);
$clourse1();
$person->getName();

// 指定作用域，可以使用$this，也可以访问
$person = new Person();
$clourse = function () {
    $this->name = 'lyj';
};
$clourse1 = $clourse->bindTo($person,Person::class);
$clourse1();
$person->getName();

// 更快捷的调用
$person = new Person();
$clourse = function () {
    $this->name = 'lyj';
};
$clourse->bindTo($person,Person::class)();
$person->getName();

// 通过传参或引用的方式使用对象，而不要使用$this
$person = new Person();
$clourse = function ($p) {
    $p->name = 'lyj';
};
$clourse->bindTo($person,Person::class)($person); // 传参 **************************
$person->getName();

$person = new Person();
$clourse = function () use($p) {
    $p->name = 'lyj';
};
$clourse->bindTo($person,Person::class)();// 引用
$person->getName();
```

稍微高级点的用法

```php
class Person
{
    protected $calls = [];
    protected $name;
    protected $age;
    public function setUser($id, Closure $closure)
    {
        $this->calls[$id] = $closure->bindTo($this, __CLASS__);
    }
    public function getUser($id)
    {
        foreach ($this->calls as $key => $callback) {
            if ($key == $id) {
                $callback();
            }
        }
        return $this->name . '-' . $this->age;
    }
}

$person = new Person();
$person->setUser('1', function () {
    $this->name = 'lyj';
    $this->age  = '30';
});
echo $person->getUser(1);
```

```php
class DB
{
    private $query;

    public function __construct()
    {
        $this->query = new Query();
    }

    public function where($value)
    {
        if (is_callable($value)) {
            $value->bindTo($this->query, Query::class)($this->query);
        } else {
            $this->query->where($value);
        }
    }
    
    public function select()
    {
        $condition = $this->query->parse();
        echo '开始执行条件' . $condition;
    }
}

class Query
{
    private $condition = [];

    public function where($value)
    {
        $this->condition[] = $value;
    }

    public function parse()
    {
        return implode(',', $this->condition);
    }
}

$db = new DB();
$db->where(101);
$db->where(101);
$db->where(101);
$db->where(101);
$db->where(function ($query) {
    $query->where(102);
});
$db->select();
```

class Persons
{
    public static $name = 'lyj';
    public static $age = 30;

    public static function setData(Student $student)
    {
        return Closure::bind(function () use ($student) {
            $student->name = Persons::$name;
            $student->age  = Persons::$age;
        }, null, Student::class);
    }
}

class Student
{
    private $name = '';
    private $age = 0;
    public function printInfo()
    {
        echo $this->name . '-' . $this->age;
    }
}

$student = new Student();
call_user_func(Persons::setData($student));
$student->printInfo();
